# vue-music-app

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### Deploy to Surge.sh
```
yarn  deploy
```

###Build app with @electron-forge
```
yarn make
```
